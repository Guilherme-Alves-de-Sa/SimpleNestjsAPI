## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Environment Setup

Sample env file:
APP_ENV=development
APP_NAME="WeScan-V2 GraphQL API"
PORT=5005
MONGO_CONNSTR=mongodb://localhost:27017/MyDataBase # production DB
MONGO_CONNSTR_DEV=mongodb://localhost:27017/MyDataBase
MONGO_DB=MyDataBase
MONGO_USERNAME=admin
MONGO_PASSWORD=12345
AUTH_JWTEXPIRES="60000s"
AUTH_JWTSECRET="my-very-big-secret-for-users"
AUTH_SCAN_JWTSECRET="my-very-big-secret-for-scanners"
AUTH_SELFSERVICE=true
AZURE_CONNECT_STRING=""
AZURE_CONTAINER_NAME=my_container
# ELASTIC
ELASTICSEARCH_NODE=http://localhost:9200
ELASTICSEARCH_USERNAME=admin
ELASTICSEARCH_PASSWORD=12345
